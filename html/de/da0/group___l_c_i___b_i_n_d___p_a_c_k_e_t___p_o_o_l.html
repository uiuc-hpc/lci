Detailed Description

Bind a packet pool to a device.

This is only needed for explicit packet pool. Implicit packet pool (the one allocated by the runtime) is automatically bound to all devices.

Parameters
[in]deviceof type device_t; positional argument; The device to bind the packet pool.
[in]packet_poolof type packet_pool_t; positional argument; The packet pool to bind.
[in]runtimeof type runtime_t; optional argument; The runtime object.
